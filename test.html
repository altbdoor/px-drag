pxDrag

pxDrag (pixelly Dragging) is a small JavaScript for object panning or "drag and drop" within a boundary

Pan Mode

The mode is pan, where the element inside is panned around, within the boundaries of its parent element.

To prevent the element inside from spilling out, please do remember to set the parent to overflow:hidden

Move Mode

The mode is move, where the element inside is moved around, within the boundaries of its parent element.
